13  * The purpose of this class is to have a type that depends only on the value_type template parameter. As iterators need
14  * to keep a pointer to the Tree<T> class (to obtain the root node), all the iterators of this container can depend only
15  * on value_type (scary iterator) and, eventually other types not related to the container. Doing so you can assign
16  * iterators of different container type (but with the same value_type, e.g.: different allocator) to each other.
17  * REMEMBER: this is not required by the standard: you shouldn't rely on this behavior for STL containers.
18  */
